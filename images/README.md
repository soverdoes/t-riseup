# 이미지 파일 매니페스트

이 폴더에 아래 파일명으로 이미지를 저장하면 자동으로 페이지에 반영됩니다.

확장자는 `.jpg` / `.png` 모두 가능하지만 **HTML에 적힌 확장자**와 정확히 일치해야 합니다. (예: `solution-1-transport.jpg` 라고 적혀있으면 `.png` 로 저장하면 안 보임)

저장 후: 브라우저에서 `Ctrl + F5` 강력 새로고침

## 팁

- **로고**: PNG에 투명 배경 권장 (`.png`)
- **사진**: JPG가 용량 작음 (`.jpg`)
- **표시 방식**:
  - 솔루션 카드, PMS/TMS, RIDEUS 등 사진 → 잘리지 않게 영역에 **꽉 채워서** 표시 (cover)
  - 케이스 카드, 고객사 로고 → 잘리지 않게 **전체 보이게** 표시 (contain)
- 파일명이 다르거나 누락되면 회색 박스에 파일명만 표시됨 → 어디 빠졌는지 한눈에 확인 가능

---

확장자가 안 맞으면 알려주세요. HTML에서 `.png` → `.jpg` 같은 변경은 쉽게 해드릴 수 있습니다.
